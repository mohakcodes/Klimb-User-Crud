## Run Locally

Clone the project

```bash
  git clone https://github.com/mohakcodes/Klimb-User-Panel.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
